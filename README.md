# Menu Wrench
A small macOS status bar app with plugin support.

## Requirements:
It requires Dropbox to be installed.
The following folder structure should exist:
- ~/MenuWrench
  - Plugins/
    - plugin1.plugin
    - plugin2.plugin
    - ...
